# Methodology Interface

Diese Schnittstelle definiert, wie Claims geprüft, verglichen, verworfen oder
vorläufig akzeptiert werden.

Sie verbindet formale Ableitung, empirische Messung und statistische Inferenz
zu einem konsistenten wissenschaftlichen Vorgehen.

## Zweck
- Klare Regeln für:
  - Testbarkeit
  - Replikation
  - Vergleich konkurrierender Hypothesen
- Vermeidung von „anything goes“ im Forschungsmodus.
- Explizite Trennung von:
  - Entdeckung
  - Rechtfertigung
  - Entscheidung

## Liefert
- Testdesigns
- Validationskriterien
- Replikationsanforderungen
- Entscheidungs- und Abbruchregeln

## Nutzt
- Logic (Ableitbarkeit)
- Mathematics (Modelle)
- Physics (Daten)
- Probability/Statistics (Evidenz)
- Common Interfaces (Validation)
