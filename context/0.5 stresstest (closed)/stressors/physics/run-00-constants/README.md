## Physics — run-00-constants
### Epistemic Context 0.5 — Stresstest

**Domain:** Physics  
**Context:** 0.5 (stresstest)  
**Run-ID:** run-00-constants  
**Purpose:** Architectural stress test  
**Authority:** None  

---

## Zweck dieses Runs

Dieser Run dient **ausschließlich** der Überprüfung der
**epistemischen Architektur** unter **maximalem Autoritätsdruck**.

Er ist **kein** Beitrag zur Physik,
**keine** Sammlung von Fakten
und **keine** Modellübersicht.

Der Fokus liegt auf der Frage:

> **Wo erzwingt die gesellschaftliche, empirische und institutionelle
> Stellung physikalischer Aussagen implizite epistemische Autorität –
> selbst dann, wenn sie vorsichtig formuliert werden?**

---

## Warum „constants“

Naturkonstanten stellen einen Extremfall dar:

- sie gelten als „gegeben“
- sie sind formalisiert
- sie sind institutionell fixiert
- sie sind technologisch eingebettet
- sie werden ontologisch gelesen

Wenn eine Architektur
Autorität **hier** nicht verhindern kann,
kann sie es nirgends zuverlässig.

---

## Rolle dieses Runs im Forschungsprogramm

Dieser Run ist:

- **kontextspezifisch** (Context 0.5)
- **experimentell**
- **nicht bindend**
- **nicht verallgemeinernd**

Er verändert **keine** globalen Prinzipien,
führt **keine** neuen Regeln ein
und begründet **keine** Ausnahmen.

Er dient ausschließlich der **Überprüfung**,
nicht der Erweiterung.

---

## Methodik

Alle Artefakte in diesem Run folgen strikt diesem Schema:

1. Auswahl eines gesellschaftlich maximal autoritativen Claims  
2. Reduktion auf zulässige Claim-Form  
3. Identifikation von Autoritätsvektoren  
4. Prüfung gegen Guardrails (Context 0.4)  
5. Entscheidung: **zulässig** oder **STOP**  
6. Dokumentation ohne Auflösung

Es findet **keine** physikalische Diskussion statt.

---

## Enthaltene Artefakte

- `stressor_speed-of-light.md`  
  Stresstest zur Konstanz der Lichtgeschwindigkeit  
  → Ergebnis: **STOP**

Weitere Artefakte sind **nicht erforderlich**,
solange sie kein neues strukturelles Muster erzeugen.

---

## Was dieser Run nicht leistet

Dieser Run:

- erklärt keine physikalischen Theorien
- bewertet keine Messungen
- priorisiert keine Modelle
- erzeugt keine Entscheidungsgrundlagen
- ersetzt keine Fachkompetenz

Er ist **kein Wissensbestand**.

---

## Verallgemeinerbarer Befund

Der Befund dieses Runs ist **strukturell**, nicht inhaltlich:

> **Autorität entsteht unabhängig von Wahrheit,
> sobald Aussagen als unumstößlich gelten.**

Physik ist hier nur der
sichtbarste Verstärker dieses Effekts.

- keine Folgeaktionen im research-program erforderlich
