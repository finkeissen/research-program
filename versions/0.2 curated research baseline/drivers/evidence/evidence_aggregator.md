# Evidence Aggregator Spec

## Input
- Formal summaries (Logic/Math)
- Verification outputs (CS)
- Data + Inference (Physics/Probability)

## Output
- EvidenceRecords
- SupportGraph
- ConflictReports

## Regeln
- EvidenceType muss gesetzt sein
- Scope alignment muss bewertet werden (match/mismatch)
- Konflikte werden berichtet, nicht „wegoptimiert“
