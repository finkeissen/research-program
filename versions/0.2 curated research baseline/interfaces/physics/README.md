# Physics Interface

Diese Schnittstelle verbindet formale Modelle (Logik/Mathematik/Informatik) mit der empirischen Welt
durch Messung, Operationalisierung, Einheiten und Unsicherheitsanalyse.

## Zweck
- Explizite Trennung von:
  - Modell
  - Messprozedur
  - beobachteten Daten
- Vermeidung des klassischen Kurzschlusses:
  „Formel ⇒ Realität“
- Herstellung prüfbarer, reproduzierbarer Aussagen über die Welt.

## Liefert
- Operationalisierte Größen
- Messprotokolle
- Datensätze mit Unsicherheitsangaben
- Modell↔Daten-Vergleiche inkl. Gültigkeitsbereich

## Nutzt
- Mathematics Interface (Modelle, Gleichungen, Parameter)
- Computer Science Interface (Auswertung, Simulation, Datenverarbeitung)
- Common Interfaces (Uncertainty, Validation)
