# Physics Boundary

## Nicht-Ziele
- Keine metaphysischen Aussagen über „was wirklich existiert“.
- Keine Reduktion auf reine Mathematik.
- Keine algorithmische Optimierung (→ CS).

## Typische Kategorienfehler
- Modellparameter = physikalische Objekte
- Messrauschen = theoretische Unschärfe
- Korrelation = Ursache

## Verhältnis zu anderen Interfaces
- nutzt Mathematik als Modellbeschreibung
- nutzt CS zur Auswertung
- liefert empirische Constraints zurück an Modelle

## Abbruchkriterien
- fehlende Operationalisierung
- fehlende Unsicherheitsangaben
- unklare Trennung Modell/Daten
