# Metrology Interface

Diese Schnittstelle definiert Mess- und Einheiten-Standards: Dimensionen, Einheiten, Kalibrierung,
Traceability und dokumentierte Messketten.

## Zweck
- Auditierbare Messungen: Einheit + Standard + Kalibrierkette.
- Einheitliche Behandlung von Skalen und Referenzen.

## Liefert
- Unit/Dimension Records
- Calibration/Traceability Records
- Measurement-Readiness Checks

## Nutzt
- physics/measurement.md (Operationalisierung)
- common/uncertainty.md
