# Semantics Boundary

## Nicht-Ziele
- keine Ontologie-Entscheidung (was "wirklich existiert")
- keine logische Ableitung
- keine empirische Validierung

## Typische Fehler
- Begriff = Messgröße (ohne Operationalisierung)
- Definition = Entdeckung
- Gleiches Wort = gleicher Sinn (über Kontexte hinweg)

## Verhältnis zu anderen Interfaces
- Vorstufe für logic/ (saubere Symbole) und physics/metrology (saubere Messbegriffe)
