# Semantics / Concepts Interface

Diese Schnittstelle stabilisiert Begriffe, Referenz und Definitionen über alle Disziplinen hinweg.
Sie verhindert Begriffsdrift und sorgt dafür, dass Formalisierung und Messung auf klaren Konzepten aufbauen.

## Zweck
- Begriffe explizit machen (Term, Referent, Definitionstyp).
- Mehrdeutigkeit markieren statt zu verschlucken.
- Operationalisierbarkeit vorbereiten (Brücke zur Physik/Metrologie).

## Liefert
- Concept Records (Begriffe + Referenz)
- Definition Records (inkl. Definitionstyp)
- Ambiguity Reports (Mehrdeutigkeit, Drift, Konflikte)
