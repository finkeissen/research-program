# Methodology Contract

## Status
exploratory — normativ, aber explizit.

## Input
- Claims / Hypothesen
- verfügbare Modelle
- Daten / Messprotokolle
- Inferenzresultate

## Output (Artefakte)
1) **Test Specification**
- was wird getestet?
- gegen was? (Null-/Alternativhypothese, Vergleichsmodell)
- unter welchen Bedingungen?

2) **Validation Criteria**
- Erfolg / Scheitern
- Akzeptanzschwellen
- Robustheitsanforderungen

3) **Replication Plan**
- was muss reproduzierbar sein?
- unter welchen Variationen?

4) **Decision Record**
- vorläufig akzeptiert
- verworfen
- offen / weiter untersuchen

## Verantwortung
- Methodologie entscheidet:
  - welche Evidenz *reicht*
  - welche Tests *angemessen* sind
- Methodologie entscheidet nicht:
  - welche Hypothese „wahr“ ist
  - welche Forschung politisch/ethisch bevorzugt wird

## Akzeptanzkriterien
- Jede Entscheidung ist begründet
- Alternativen sind benannt
- Unsicherheit wird explizit akzeptiert oder adressiert
