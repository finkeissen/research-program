# Probability & Statistics Interface

Diese Schnittstelle formalisiert Unsicherheit, Zufälligkeit und Inferenz
über alle Disziplinen hinweg.

Sie ist kein reines „Hilfsmodul“, sondern eine eigenständige Grundlagen-
schnittstelle zwischen Modell, Daten und Entscheidung.

## Zweck
- Explizite Behandlung von Unsicherheit statt impliziter Intuition.
- Trennung von:
  - Zufälligkeit
  - Unwissen
  - Modellannahmen
- Formale Bewertung von Evidenz und Vergleich konkurrierender Hypothesen.

## Liefert
- Wahrscheinlichkeitsmodelle
- Inferenzresultate (Schätzung, Test, Posterior)
- Unsicherheitsquantifizierung
- Gütemaße und Modellvergleiche

## Nutzt
- Physics Interface (Daten, Messfehler)
- Mathematics Interface (Wahrscheinlichkeitsräume)
- Computer Science Interface (Simulation, Sampling)
- Common Interfaces (Uncertainty, Validation)
