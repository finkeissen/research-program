# Probability & Statistics Contract

## Status
exploratory — konzeptuell stabil, methodisch offen.

## Input
- Datensätze (mit Metadaten)
- Hypothesen / Modelle
- Annahmen über Zufälligkeit / Rauschen
- Inferenzziel:
  - Schätzung
  - Test
  - Prognose
  - Vergleich

## Output (Artefakte)
1) **Probability Model**
- Zufallsvariablen
- Verteilungen
- Abhängigkeiten

2) **Inference Result**
- Punktschätzung
- Intervall (Konfidenz/Kredibel)
- Posterior / Likelihood

3) **Model Assessment**
- Gütemaße
- Residuen
- Sensitivitätsanalyse

4) **Decision-Relevant Summary**
- Evidenzstärke
- Robustheit gegenüber Annahmen
- bekannte Limitationen

## Verantwortung
- Statistik entscheidet:
  - wie stark Daten eine Hypothese stützen
- Statistik entscheidet nicht:
  - ob eine Hypothese „wahr“ ist
  - welche Hypothese *normativ* bevorzugt werden soll

## Akzeptanzkriterien
- Alle Verteilungen/Modelle sind explizit angegeben
- Annahmen über Unabhängigkeit/Stationarität sind markiert
- Ergebnisse enthalten Unsicherheitsangaben
