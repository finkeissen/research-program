# Mathematics Primitives

## Objekte
- **Set / Collection**
- **Element**
- **Structure** (z. B. (X, +, ·), (V, ||·||), (G, ∘))
- **Function / Map**
- **Relation**
- **Order / Topology / Metric** (falls relevant)
- **Parameter**

## Aussagen
- Definition
- Lemma
- Theorem
- Corollary
- Proposition

## Beweisformen (nicht exklusiv)
- direkter Beweis
- Induktion
- Widerspruch
- Konstruktion
- Reduktion

## Metamarker
- `exists`
- `unique`
- `for_all`
- `bounded`
- `invariant_under`
- `approximate`

## Scope / Gültigkeit
- definierter Raum (z. B. endlich, kompakt, diskret)
- Regularitätsannahmen (stetig, differenzierbar, messbar)
