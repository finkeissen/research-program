# Evidence Interface

Diese Schnittstelle beschreibt Evidenz als Artefakt: Evidenztypen, Stärken, Konflikte,
und nachvollziehbare Zusammenfassungen für Entscheidungen (Methodology).

## Zweck
- Evidenz sichtbar und vergleichbar machen.
- Konflikte nicht „lösen“, sondern als Reports exportieren.

## Liefert
- EvidenceRecords
- ConflictReports
- SupportGraphs (Claim ↔ Evidence)
