# Physics Primitives

## Objekte
- **Physical Quantity** (mit Einheit)
- **Measurement**
- **Instrument**
- **Experiment / Observation**
- **Data Point / Data Set**
- **Parameter**
- **State** (modellabhängig)

## Eigenschaften
- Einheit / Dimension
- Skala (nominal, ordinal, intervall, ratio)
- Auflösung
- Reproduzierbarkeit

## Aussagen
- Messwert
- Mittelwert / Streuung
- Korrelation
- Abweichung vom Modell

## Marker
- `measured`
- `derived`
- `calibrated`
- `model-dependent`
- `approximate`
