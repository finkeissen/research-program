# Measurement (Physics)

## Grundprinzip
Keine physikalische Aussage ohne explizite Messdefinition.

## Operational Definition
Für jede Größe:
- Name
- Messprozedur
- Gerät / Sensor
- Einheit
- Gültigkeitsbereich

## Messarten
- direkte Messung
- indirekte Messung (über Modell)
- Beobachtung vs. Intervention

## Unsicherheit
- statistisch (Rauschen, Stichprobe)
- systematisch (Bias, Drift)
- externe Effekte (Umwelt, Kopplungen)

## Kalibrierung
- Referenzstandard
- Zeitpunkt
- bekannte Abweichungen

## Dokumentationspflicht
Messung ohne Dokumentation ⇒ **nicht gültig** für Weiterverarbeitung.
