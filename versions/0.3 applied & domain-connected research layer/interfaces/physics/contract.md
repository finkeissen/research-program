# Physics Contract

## Status
exploratory — empirisch streng, aber disziplinübergreifend abstrahiert.

## Input
- mathematische Modelle (Gleichungen, Parameter, Strukturen)
- zu prüfende Hypothesen / Vorhersagen
- Kontextannahmen (z. B. Idealisierungen)

## Output (Artefakte)
1) **Operational Definitions**
- welche Größe wird wie gemessen?
- Messvorschrift, Gerät, Einheit

2) **Measurement Protocol**
- Versuchs-/Beobachtungsaufbau
- Randbedingungen
- Wiederholungsregeln

3) **Data Record**
- Rohdaten
- abgeleitete Größen
- Metadaten

4) **Uncertainty & Error Model**
- statistische Unsicherheit
- systematische Effekte
- Kalibrierung

5) **Model–Data Comparison**
- Fit, Abweichung, Residuen
- Gültigkeitsbereich
- bekannte Brüche/Anomalien

## Verantwortung
- Physik entscheidet:
  - was *gemessen* wurde
  - mit welcher Genauigkeit
- Physik entscheidet nicht:
  - ob ein Modell „die Wahrheit“ ist
  - ob Alternativmodelle besser sind (das ist Methodologie)

## Akzeptanzkriterien
- Jede Messgröße hat:
  - Einheit
  - Messvorschrift
  - Unsicherheitsangabe
- Modelle sind klar von Daten getrennt
