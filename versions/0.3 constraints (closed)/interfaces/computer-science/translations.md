# Computer Science Translations

## Mathematik → Algorithmik
Kennzeichnung:
- konstruktiver Beweis → direkter Algorithmus
- Existenzbeweis → Such-/Entscheidungsproblem
- Stetige Strukturen → Diskretisierung nötig

## Algorithmus → Erklärung
- Ziel des Algorithmus
- zentrale Idee
- bekannte Grenzen

## CS → Operatives System (mms)
- klare Input/Output-Spezifikation
- deterministisch vs. stochastisch
- Logging-/Tracing-Punkte
- Validierbare Properties

## Übergabe-Artefakte (vorläufig)
- `AlgorithmRecord`
- `DataModelRecord`
- `ComplexityRecord`
- `VerificationRecord`
