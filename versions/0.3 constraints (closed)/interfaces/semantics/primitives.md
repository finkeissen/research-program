# Semantics Primitives

## Objekte
- Term (Ausdruck)
- Sense (Bedeutungsvariante)
- Referent (Worauf zeigt der Term in einem Kontext?)
- Definition (Festlegung)
- Context/Scope (Domäne, Bedingungen)

## Definitionstypen
- stipulative: Festlegung (Konvention)
- descriptive: Beschreibung üblicher Verwendung
- operational: Mess-/Prozedurdefinition
- theoretical: Begriff im Rahmen einer Theorie

## Marker
- `ambiguous`
- `drifting`
- `undefined`
- `operationalizable?`
