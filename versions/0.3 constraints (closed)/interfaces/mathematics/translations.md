# Mathematics Translations

## Claim → Mathematical Statement
Mapping enthält:
- claim_id
- verwendete Definitionen
- Annahmen/Axiome
- mathematische Aussage (formal/informal)
- Gültigkeitsbereich

## Beweis → Strukturierte Erklärung
- Ziel
- Annahmen
- Hauptschritte
- Abhängigkeiten (Lemmas)

## Mathematik → Informatik
Kennzeichnung:
- konstruktiv vs. nicht-konstruktiv
- exakt vs. approximativ
- diskretisierbar ja/nein

## Mathematik → Physik
Kennzeichnung:
- Modelltyp (idealisiert, effektiv, asymptotisch)
- Parameter mit physikalischer Bedeutung?
- explizite Gültigkeitsgrenzen
