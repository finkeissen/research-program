# Methodology Translations

## Claim → Testdesign
- präzise Hypothese
- relevante Variablen
- Kontrollbedingungen

## Evidenz → Entscheidung
- Zusammenfassung der Ergebnisse
- Bewertung der Unsicherheit
- Alternativerklärungen

## Entscheidung → Forschung
- nächste Experimente
- Modellrevision
- Abbruch oder Fokuswechsel

## Übergabe-Artefakte (vorläufig)
- `TestSpecRecord`
- `ValidationRecord`
- `DecisionRecord`
